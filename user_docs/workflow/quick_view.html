QuickView

VFXやAnimationのアーティストは、xSTUDIOのPlaylistsやMedia Listを操作することなく、個別のMedia項目(最新のPlayblastレンダリングなど)を単体で素早く確認したい場合があります。このようなワークフロー機能を提供するため、xSTUDIOには、単一のMedia項目のみを表示する、独立した軽量のViewportインスタンスである「QuickView」プレイヤーウィンドウが用意されています。これらのウィンドウは、main interface、command line、またはPython APIから非常に素早く起動できます。

Note

QuickView機能で新しいメディアをロードする際は、xSTUDIOインスタンスがすでに起動していることが重要です。これにより、QuickViewウィンドウを非常に素早く起動できるようになります。この機能を利用するには、デスクトップ上のどこかでフル機能のxSTUDIOインスタンスが常に起動している必要があります。

Note

QuickView機能でロードされた新しいメディアは、常にメインのxSTUDIOセッションに追加されます。これには、QuickViewウィンドウを閉じた後でも、いつでもそのMedia項目に戻って再確認できるというメリットがあります。

QuickView from the Command Line

xSTUDIOを実行するためのターミナル/シェルのショートカットやエイリアスを設定している場合は、--quick-viewのcommand lineフラグを使用することで、xSTUDIOをQuickViewウィンドウに簡単にロードできます。

QuickView a media item from a terminal (assuming you have a shortcut/alias to the xstudio binary)
xstudio /Users/jane/data/renders/project_001/shot_f0010/renders --quick-view

Note

QuickView機能を使用するにはxSTUDIOのセッションがすでに起動している必要がありますが、メインのxSTUDIOウィンドウは最小化されているか、あるいは画面外(別のデスクトップワークスペースなど)に配置されていても構いません。

QuickView from the Python API

独自のツールと統合するためにxSTUDIOのAPIを使用し、MediaをQuickViewウィンドウにロードしたいユーザーの場合、Python APIを使用して簡単にこれを行うことができます。

Launch a quickview player
XSTUDIO = Connection(auto_connect=True)
import glob
renders = glob.glob("/Users/joe/reference/water/*wave*.mp4*")
# load multiple files into a single QuickView using 'Grid' compare
XSTUDIO.api.session.quickview_media(renders, compare_mode="Grid")

QuickView from the Interface

必要に応じて、インターフェースからMediaをQuickViewすることもできます。Media List Panelで1つまたは複数のMedia項目を選択し、右クリックのコンテキスト/詳細（More）メニューから「QuickView Selected」を選択します。
