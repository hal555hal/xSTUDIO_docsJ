Python ‘Snippets’

Pythonのsnippetは、SnippetsメニューまたはMedia ListメニューからxSTUDIO内で実行できる、自己完結型のPython scriptです。この目的は、便利なPython scriptsをxSTUDIO環境に追加し、特定のタスクの自動化、playlistsのbuild、あるいはPython APIを介したSessionとのその他の相互作用を行うための便利な方法を提供することです。

Creating a Snippet

snippetを作成するには、xSTUDIOがsnippetsをスキャンする場所のいずれかに、拡張子が「.py」のファイルを作成するだけです。これらの場所は、ファイルシステムパスのリストであるsnippets pathの環境設定(Preference)によって制御されます。デフォルトでxSTUDIOがチェックするパスの1つは、$HOMEフォルダ内のxStudio/snippetsです。そのため、ホームスペースにxStudioフォルダを作成し(大文字と小文字に注意してください)、その中にsnippetsフォルダを作成すれば、その場所にsnippetsを配置してxSTUDIOに検出させることができます。

たとえば、このスクリーンショットでは、Preferences Managerを介してパス/user_data/ted/snippetsがsnippets pathに追加されています…

ここで、私のホームスペースにあるデフォルトの場所C:\Users\tedwaine\xStudio\snippetsにmy_test_snippet.pyというファイルを作成し、Snippetsメニューの下にあるReloadを押すと、私のSnippetsメニューは以下のようになります：

ここで「My Test Snippet」をクリックすると、my_test_snippet.pyファイル内のコードがxSTUDIOのpython interpreter内で実行されます。

Note

Media Listの右クリックによるContext/Moreメニューにsnippetsを表示させることもできます。snippets用にスキャンされるフォルダのいずれかの内部に、mediaという名前の別のフォルダを追加します。mediaフォルダ内にあるすべての.pyファイルは、Media Listメニュー内のSnippetsサブメニューに表示されます。

Example Snippet

snippetからは、お好みのあらゆるpythonコードを実行できます。以下は、現在表示されているSequence内にあるすべての選択されたクリップの色をランダムにするシンプルな例です(ViewportでSequenceが表示されている場合)。

my_test_snippet.py
from xstudio.api.session.playlist.timeline import Timeline, Clip
import random
import colorsys

def random_clip_colour(item=XSTUDIO.api.session.viewed_container):
    if isinstance(item, Timeline):
        for i in item.selection:
            if isinstance(i, Clip):
                c = random.randrange(0, 255)

                rgb = colorsys.hsv_to_rgb(c * (1.0/255.0), 1, 1)
                i.item_flag = "#FF" + f"{int(rgb[0]*255.0):0{2}x}"+ f"{int(rgb[1]*255.0):0{2}x}"+ f"{int(rgb[2]*255.0):0{2}x}"

random_clip_colour()
