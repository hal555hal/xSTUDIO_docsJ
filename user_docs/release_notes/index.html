Release Notes

v1.3.0

Overview

xSTUDIOは、VFX、アニメーション、およびポストプロダクションのプロフェッショナルによって、プロフェッショナルのために設計された高性能な再生・レビューツールです。このアプリケーションは、大量のメディアファイルのコレクションをロードして再生できます。効率的な再生エンジンにより、幅広いファイルフォーマットとエンコーディングの高解像度画像フォーマットをすばやくロードして再生できます。直感的なツールにより、プレイリストやプレイリスト内のメディアサブセットを作成・整理して、インタラクティブなレビューセッションや画像・ビデオのリファレンスライブラリを構築できます。マルチトラックのタイムライン編集インターフェースにより、単純なものから複雑なものまで、編集のロードや作成を行う機能が提供されます。xSTUDIOのPythonおよびC++ APIは、プロダクションパイプラインやアセット管理システムとの統合やカスタマイズのための強力なツールを提供します。xSTUDIOは、dailies reviewからVFX shot review、gradingやcolour management、クライアントへのプレゼンテーションなどに至るまで、制作プロセスにおいてさまざまな方法で使用できる柔軟なツールとして設計されています。

New in this version (v1.3.x)

  • Filesystem Browserインターフェース

  • ユーザー設定可能なHotkeys

  • AnnotationドローオーバーにおけるDodgeおよびBurnツール

  • アニメーションドローオーバーのvisualisation用「Onion-skinning」plugin

  • 画像(フレーム)ベースのロードのさらなるoptimisation

  • マスクレンダリングおよびAPIの向上

  • Mac、Linux、およびWindows用のbuildワークフローの向上

  • 更新されたドキュメントおよびリリースノート

  • コードのクリーンアップ：「Clang tidy」および-Werror(gcc)をパス

Features available since v1.2.x

  • Blackmagic Designs Decklinkカードを介したSDI出力

  • 3Dアプリケーションへのインポート用に、annotationsを画像またはGrease Pencilパッケージとしてエクスポートする機能

  • デモ/リファレンス用パイプライン統合plugin(開発者がxSTUDIOとのパイプライン統合を記述する際のモデルとなることを想定)

  • xSTUDIO内でPDFドキュメントをフレームベースのメディアアイテムとして表示するためのPDFリーダーplugin

  • Ninja buildシステムを使用した、Windows向けの劇的に高速なbuildサポート

  • プレイリスト、タイムライン、コンタクトシートをレンダリングし、好みのffmpegエンコーディングパラメータで迅速かつ容易にビデオを出力する機能

  • ツールバー、右クリックのビューポートメニュー、またはHotkeysショートカットを介したメディアFPSの設定機能

  • コミュニティからのフィードバックに着想を得た多数のAPI拡張

xSTUDIO Highlights

  • 効率的で高性能な再生エンジン

  • 幅広いメディアファイルフォーマットとの互換性

  • 柔軟で高度にカスタマイズ可能なユーザーインターフェース

  • シンプルで直感的なプレイリスト駆動型のセッションワークフロー

  • マルチトラックNLEタイムラインインターフェース

  • OpenColorIO統合による正確なcolour management

  • ノート入力ワークフロー：テキストのみ、ドローオーバー、またはその両方

  • gradingまたはgradingノート作成用のカラーコレクションツール

  • スクリプティング、plugin拡張、およびリモートコントロール用のPython API

  • 高性能なplugin拡張およびグラフィックスオーバーレイ用のC++ API
