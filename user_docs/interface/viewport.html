The Viewport

Viewport は画像が表示される panel であり、当然ながら最も多くの時間を費やして注目する場所です！一定のカスタマイズが可能であり、これは Context/More メニュー (Viewport 内でマウスを右クリックするか、Viewport の最右上にある「More」ボタンをクリックする) から行うことができます。

Note

Viewport アクションバーにあるすべてのボタンには、関連するホットキーショートカットが割り当てられています。そのため、これらのショートカットに習熟していれば、関連する機能へのアクセスを損なうことなくアクションバーを非表示にすることができます！

Interacting with the Viewport

3 ボタンマウスを使用して、画像をより詳細にインスペクトできます。中央ボタンを長押ししてドラッグすると、Viewport 内で画像をパンできます。キーボードの CTRL キーを押しながらマウスの中央ボタンで左右にドラッグすると、画像を基準とした中央ズームイン/ズームアウトを行えます。

Viewport アクションバーには、ズームモードとパンモードを切り替えるためのボタンが用意されており、これらをオンにすると、プライマリ (左) マウスボタンのドラッグで Viewport 内の画像をズームイン/ズームアウトまたはパンできるようになります。画像をズーム/パンした後は、Fit ホットキー (デフォルトキー : F) を押すことでフィットモードに戻ります。

xSTUDIO の環境設定には、マウスホイールの挙動を設定するオプションがあります。「Zoom Viewer」に設定されている場合は、マウスホイールを回転させてビューア内の画像をズームイン/ズームアウトできます。あるいは、中央のマウススクロールホイールを使用して、フレームを前後にシャッフル (コマ送り) することも可能です。

メイン (左) マウスボタンをクリックして左右にドラッグすると、プレイヘッドがフレームを前後にスクラブします。この操作のマウス感度も、ユーザー環境設定で設定できます。

Note

ビューアを「リセット」して、画像が自動フィットされ、露出がノーマライズされ、チャンネル設定が RGB に戻り、再生レートが 1.0 になるようにするには、Viewport アクションバーの Reset ボタンを押すか、「CTRL+R」のホットキーシーケンスを使用します。
