Transport Controls and Toolbar

Toolbar とトランスポートコントロールは、Viewer のすぐ下にあります。Toolbar には、ビューアの外観や挙動を変更するためのコントロールが用意されています :

Transport Controls

xSTUDIO のトランスポートコントロールを使用すると、再生の開始/停止や、Sequence 内のメディア項目のジャンプ（移動）を行えます。各ボタンの機能はユーザーにとって概ね馴染みのあるものですが、以下の点に注意してください :

  • タイムコード/現在のフレーム表示をクリックすると、表示を変更するオプションが表示されます。

  • 「Frames from timecode」を選択すると、タイムコードが絶対タイムコードフレームとして表示されます。例えば、24fps のメディア項目のタイムコードが 00:00:10:10 の場合、絶対フレームは 254 と表示されます。

  • タイムラインバー内をクリックしてスクラブすることで、現在のフレームを設定できます。

  • スナップショットツールを使用すると、現在のフレームを静止画 (画面上のアノテーションを含む) として書き出すことができます。

  • タイムバー領域において、緑色のブロックはデコードが完了し、再生用にキャッシュに保存されているフレームを示します。画像キャッシュのサイズを調整するには、xSTUDIO preferences を参照してください。

  • フレームが利用できない場合 (例えば、長さの異なる 2 つのメディア項目を A/B 比較しており、長さが短い方の項目を表示している場合など)、タイムラインにオレンジ色のバーが表示されます。

xSTUDIO Toolbar

xSTUDIO の Toolbar には、メディアの再生や表示に関するさまざまな設定を操作できるボタンが多数用意されています。Toolbar の各ボタンは、ユーザー設定によって表示/非表示を切り替えることができるため、必要に応じて必要なボタンのみを配置することが可能です。また、必要であれば Toolbar 自体を完全に非表示にすることもできます。Toolbar ボタンで制御される一部のステータスは、xSTUDIO のホットキー経由でも調整できます。Toolbar の各要素の概要は以下の通りです :

HUD

詳細は xSTUDIO HUD のページを参照してください。ここをクリックすると、HUD (ヘッドアップディスプレイ) のオン/オフを切り替えることができます。HUD は複数の要素で構成されており、それぞれの要素を独立してアクティブ/非アクティブに切り替えることが可能です。

Rate

マウスボタンを長押ししたまま左右にドラッグ (スクラブ) して、再生「レート」を変更します。ダブルクリックするとリセットされます。これはプレイヘッドの速度に適用される乗数であり、再生速度を細かくコントロールして遅くしたり速くしたりできます。

Exposure

クリックして左右にドラッグすると、Viewer の露出を調整できます。ダブルクリックするとリセットされます。露出は、キーボードの「E」ホットキーを押しながら Viewport 内でマウスを左右にドラッグすることでも調整可能です。

Gamma

露出と同様の操作ですが、Viewer のガンマをコントロールします。Viewport 内でガンマをスクラブするためのデフォルトのホットキーは「Y」です。

Fit

Viewport の「Fit」モードを設定します。Viewport の形状に合わせて画像が自動スケーリングされるように制御します。

  • Width : 画像の幅が Viewer の幅にぴったり収まるようにFitさせます

  • Height : 画像の高さが Viewer の高さにぴったり収まるようにFitさせます

  • Best : 画像全体が常に Viewer 内に表示されるように、幅または高さへのFitを自動選択します

  • Fill : Viewer が常に画像で満たされるように、幅または高さへのFitを自動選択します

  • 1:1 : 画像の 1 ピクセルがディスプレイデバイスの 1 ピクセルにマッピングされるように、画像をスケーリングして中央に配置します

画像をパンまたはズームすると、「Fit」モードは「Off」に切り替わり、自動スケールモードから抜けて Viewer が現在のパン/ズーム設定を維持していることを示します。「Fit」ボタンをダブルクリックすると、最後に使用した「自動Fit」の Fit モード、またはカスタムのパン/ズーム設定に切り替わります。

Channel

Viewport に表示する単一のカラーチャンネル (R、G、B または Luminance) を選択できます。デフォルトのホットキーは R、G、B、および CTRL+L (Luminance 用) です。

Mirror

画像全体を反転 (フリップ / フロップ、またはその両方) させて、Viewport 全体をトランスフォームします。(ミラーモードを切り替えるデフォルトのホットキー : SHIFT + F)

Compare

複数選択されたメディアの比較方法を設定します。詳細については「Playhead Controls」のセクションを参照してください。

Display

この設定は、画像を Viewport 領域にどのように自動 Fit させるかを決定します。

  • Width : 画像の幅が Viewer の幅にぴったり収まるようにFitさせます

  • Height : 画像の高さが Viewer の高さにぴったり収まるようにFitさせます

  • Best : 画像全体が常に Viewer 内に表示されるように、幅または高さへのFitを自動選択します

  • Fill : Viewer が常に画像で満たされるように、幅または高さへのFitを自動選択します

  • 1:1 : 画像の 1 ピクセルがディスプレイデバイスの 1 ピクセルにマッピングされるように、画像をスケーリングして中央に配置します

画像をパンまたはズームすると、「Fit」モードは「Off」に切り替わり、自動スケールモードから抜けて Viewer が現在のパン/ズーム設定を維持していることを示します。「Fit」ボタンをダブルクリックすると、最後に使用した「自動 Fit」の Fit モード、またはカスタムのパン/ズーム設定に切り替わります。

Channel

R/G/B/A チャンネルの単独表示、または Luminance（輝度）表示に使用します。

Compare

複数選択されたメディアの比較方法を設定します。詳細については、これらのページの Compare Modes セクションを参照してください。

Display

物理的なディスプレイデバイスのカラースペースを選択します。アクティブ・カラーマネジメントを使用することで、モニター/プロジェクターの特性を補正し、表示されている画像が正確に表現されるようになります。

View

このメニューから OCIO ビュープロファイル (通常はグレード/ルック + フィルムエミュレーション LUT) を設定します。詳細については Colour Management セクションを参照してください。

Source

ディスク上に複数のソースを持つ Media 項目について、このボタンを使用して画像とオーディオの両方のアクティブなソースを選択できます。

複数の Stream、Part、または Layer を持つ画像 (EXR や、.mov などの一部のコンテナ化された画像フォーマットなど) の場合、こちらのポップアップメニューからアクティブにする Stream や Part を選択できます。必要に応じて、ポップアップから Stream/Part/Layer を選択できるフローティングウィンドウを開くことも可能です。

Media 項目および Media ソースの詳細については、xSTUDIO Media Items を参照してください。

Configuring the xSTUDIO Toolbar

Viewport の More/Context メニュー (Viewport 内で右クリックするか、Viewport パネルの右上にある More ボタンをクリックします) から Toolbar サブメニューにアクセスし、Toolbar 項目のアクティブ/非アクティブを切り替えることで、ご自身の Toolbar をすっきりと整理したり、機能を充実させたりすることができます。
